# Mjukvara

## Design, programmering och drift i ett ekosystem

Rockwell Automations mjukvaruportfölj ger dig verktygen för hela livscykeln - från design och simulering till programmering, driftsättning, visualisering och underhåll. Studio 5000 är den gemensamma utvecklingsmiljön för alla Logix-kontroller, medan FactoryTalk-sviten hanterar allt från HMI och SCADA till historik, fjärråtkomst och prediktiv analys. Emulate3D ger dig möjlighet att simulera och virtuellt driftsätta din automation innan den byggs fysiskt. För bredare produktionsbehov finns även molnbaserade system för MES och underhållsplanering. Connected Components Workbench ger kostnadsfri programmering för Micro800-kontroller, PowerFlex-omriktare och PanelView-paneler. Vi hjälper dig att navigera mjukvarulandskapet och hitta rätt licensmodell för dina behov.

Utvecklingsverktyg — Från programmering till virtuell idrifttagning

**Studio 5000 Logix Designer** är Rockwell Automations primära programmeringsverktyg för alla Logix-controllers. Miljön samlar PLC-logik, rörelsestyrning, drivsystemskonfiguration (PowerFlex, Kinetix) och säkerhetsprogrammering i ett gemensamt projekt. Stöder ladder diagram, structured text, function block diagram och sequential function chart.  
**Studio 5000 Architect** designar och dokumenterar systemarkitekturen - styrenheter, nätverk och I/O-konfiguration - innan programmeringen börjar.  
**FactoryTalk Design Studio** är Rockwells molnbaserade Logix-designverktyg. Webbläsarbaserad åtkomst utan installation, fleranvändarsamarbete med inbyggd versionshantering, och en AI Copilot som assisterar vid kodgenerering, felsökning och dokumentation.  
**FactoryTalk Logix Echo** emulerar Logix-styrenheter i mjukvara för offlinetest, utbildning och virtuell idrifttagning.  
**Emulate3D** skapar fysikbaserade 3D-modeller av maskiner och produktionslinjer som ansluts till verkliga styrsystemsprogram – antingen via Logix Echo eller direkt mot fysiska styrenheter. Det möjliggör virtuell idrifttagning, genomflödessimulering och operatörsträning i virtuell miljö. Importerar CAD-modeller från SolidWorks, Inventor och Creo.  
**Motion Analyzer** är ett kostnadsfritt molnbaserat dimensioneringsverktyg för servo- och drivsystem, med tillgång till hela Kinetix-katalogen inklusive växlar från teknikpartner.

Connected Components Workbench — Kostnadsfri programmering för kompakta system

Connected Components Workbench (CCW) är Rockwells kostnadsfria utvecklingsmiljö för kompakta styrsystem. Programvaran samlar programmering och konfiguration av Micro800-kontroller, PowerFlex 520-seriens frekvensomriktare, PanelView 800-terminaler och Kinetix 5100-servoförstärkare i ett och samma verktyg.  
  
CCW erbjuder ladder diagram, structured text och function block diagram, samt inbyggd simulering för offlinetest. Verktyget riktar sig till maskinbyggare och OEM:er som bygger fristående maskiner utan behov av Logix-plattformen.  

* ✓ Kostnadsfritt: ingen licenskostnad, laddas ner direkt från Rockwell Automations webbplats.
* ✓ PLC, VFD, HMI och servo i ett verktyg - Micro800, PowerFlex 520, PanelView 800 och Kinetix 5100 i samma programvara.

Visualisering — Mjukvara för visualisering, HMI, och SCADA

Rockwell Automation erbjuder två parallella HMI/SCADA-plattformar, den beprövade FactoryTalk View-familjen och den nya FactoryTalk Optix.  
**FactoryTalk View ME** körs på PanelView Plus-terminaler och ger operatörer realtidsöverblick med larm, trender och navigering direkt vid maskinen.  
**FactoryTalk View SE** är en skalbar SCADA-lösning med klient/server-arkitektur för centraliserad processövervakning. Hanterar distribuerade HMI-klienter, larm, trender och historikintegrering. Integreras med ThinManager för centraliserad thin client-hantering. Används i process-, batch- och diskret tillverkning.  
**FactoryTalk Optix** är nästa generations HMI- och dataplattform. Webbbaserat projekteringsverktyg med fleranvändarsamarbete och versionshantering, C#-scripting för utbyggbarhet, och stöd för OPC UA, REST API och MQTT utöver EtherNet/IP. Kör på Windows och Linux. Optix samlar HMI med edge-datainsamling som kan skicka kontextualiserade maskindata vidare till molntjänster och analysverktyg. OptixPanel-terminaler levereras med förinstallerad Optix och Remote Access.
